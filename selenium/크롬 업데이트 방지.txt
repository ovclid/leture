

ㅇ 크롬 업데이트 방지
참고 : https://rgy0409.tistory.com/2301
1. 윈도우즈키 + r -> msconfig -> "Google 업데이트 서비스" 관련 모두 체크 해제 -> 다시 시작하지않고 끝내기(다음 작업을 위해)
2. 윈도우즈키 + s -> 작업스케줄러 -> 작업 스케줄러 라이브러리 -> GoogleUpdate 관련 모두 사용안한 ("상태" 속성에서 마우스 우클릭 후 선택)
3. C:\ > Program Files (x86) > Google > Update -> GoogleUpdate.exe 파일의 파일명을 변경

4. 1~3 작업 완료 후 크롬 실행 -> Chrome 정보 -> 업데이트 실행


ㅇ 크롬 다운그레이드
1. 이전 버전 크롬 다운로드 (https://google-chrome.en.uptodown.com/windows/versions)
2. 현재 설치 버전 삭제
3. 이전 버전 크롬 설치
